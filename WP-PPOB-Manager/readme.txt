== Description ==
Plugin PPOB lengkap untuk WordPress dengan integrasi WooCommerce. Mendukung pulsa, PLN, BPJS, PDAM, dan voucher game.

== Installation ==
1. Upload plugin ke folder wp-content/plugins/
2. Aktifkan plugin melalui menu Plugins
3. Masukkan API key Digiflazz di menu PPOB Manager > Settings
4. Sinkronkan produk dan mulai berjualan
